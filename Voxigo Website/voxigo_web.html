Empowering Communication for Neurodivergent Children

Voxigo helps children express themselves with ease using pictograms, AI-powered sentence helpers, games, music, and more. Join us in creating a supportive, joyful environment for learning and growth.

How It Works

Voxigo simplifies communication through intuitive features like pictograms and AI-powered assistance. Designed to empower neurodivergent children, it provides a seamless and enjoyable experience for both kids and parents.

Getting Started

To begin using Voxigo:

  • Parents need to create an account first, either by signing up manually or using Google Sign-in for quick access.
  • Once logged in, parents can create a secure login for their child.
  • The child can then start using Voxigo's interactive features for communication and learning.

Frequently Asked Questions

What is VoxigoAAC?

VoxigoAAC is an assistive communication app for nonverbal and neurodivergent children, featuring an AI-powered AAC grid, customizable communication buttons, and fun activities.

How does VoxigoAAC work?

Parents sign up, create a child profile, and kids use the AAC grid to communicate. Parents receive insights, and kids can engage in activities like music, coloring, and breathing exercises.

Is VoxigoAAC free?

Yes! VoxigoAAC is completely free with no in-app purchases or hidden fees.

Can I delete my profile?

Yes, you can delete your profile at any time, and all personal data will be permanently removed.

Can I delete my child's profile?

Yes, parents have full control over their child's profile and can delete it permanently if needed.

Can I control which features my child can access?

Yes! Parents can enable or disable specific features, set time limits, and restrict access to certain activities.

How does the AI assistant help with communication?

Voxigo's AI assistant suggests words and phrases based on usage patterns, helping children communicate more effectively.

Can kids customize the AAC grid?

Yes, both kids and parents can add new words, images, and phrases to personalize their communication experience.

Can parents track their child's emotions and communication?

Yes, VoxigoAAC provides insights into daily, weekly, and monthly communication history, allowing parents to monitor patterns and receive AI-generated suggestions.
